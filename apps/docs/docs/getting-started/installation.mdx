Formik-form-builder consists of a set of npm [packages](https://github.com/facebook/docusaurus/tree/main/packages).

:::tip[Bright side]

Use formik-form-builder to handle forms easily

Try it out [now](https://github.com/facebook/docusaurus/tree/main/packages)
:::

---

## Requirements

<ul>
  <li>
    [Node.js](https://nodejs.org/) version 18.0 or above (which can be checked
    by running node -v). You can use nvm to manage multiple Node.js versions on
    a single machine.
    <ul>
      <li>
        When installing Node.js, it is recommended to check all checkboxes
        related to dependencies.
      </li>
    </ul>
  </li>
</ul>

## Scaffold it

  </TabItem>
  <TabItem value="yarn" label="Yarn">
    ```bash 
    yarn add formik-form-builder
    ```
